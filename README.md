# kavem_proyect_documentation


EL siguiente repositorio tiene como objetivo documentar el trabajo de todos en el proyecto de KAVEM, teniendo como resultado
diagramas entidad relacion, diagramas de casos de uso, mockups, diagramas de navegacion, scenary cards por cada 
entidad en el diagrama de caso de uso y ademas avances del codigo de la aplicacion donde tenemos los sprint.


La metodologia que estamos usando en el proyecto es en base a la metologia scrum, donde tenemos sprints, y por cada uno
desarrollaremos distintos apartados del codigo de la aplicacion.


Nuestro objetivo es desarrollar una aplicacion web, que gestione control de inventario, clientes y ademas genere cotizaciones
y posteriormente pedidos.



# Trabajo

El trabajo esta organizado de forma distinta dependiendo de quien haga que cosa, y se divide de forma diferente en los sprints
debido a que por unidad cambiamos de scrum-master.

Este repositorio solo tiene como objetivo la subida y descarga de documentacion.
